# 프로젝트 시작하기

이 프로젝트(Mantine + Vite + React)를 처음 실행해 시작 화면까지 확인하는 절차입니다.
수강생은 `node_modules`도 없는 상태로 시작합니다. 
아래 순서대로 진행하세요.

## 절차

1. **의존성 설치** — 터미널에서 `npm install` 실행.
   - `node_modules` 폴더가 이미 있으면 이 단계는 건너뜁니다.
2. **개발 서버 실행** — `npm run dev` 실행.
   - 서버는 계속 떠 있어야 하므로 백그라운드로 실행하고 종료하지 마세요. ()
3. **화면 확인** — 서버가 뜨면 터미널에 출력된 주소(기본 `http://localhost:5173`)를 수강생이 열수 있도록 링크를 안내합니다.

## 주의
- 수강생에게 계획안내는 불필요합니다. 
- 개발서버 실행을 브라우저 테스트하지 말아주세요.
- 권장 Node 버전은 `.nvmrc` 기준(24.16.0)입니다.
- 설치나 실행 중 에러가 나면, 에러 메시지를 수강생에게 그대로 보여주고 멈춘 뒤 다음 안내를 기다립니다.
- 설치 명령은 항상 `npm`을 사용합니다(yarn 아님).
